# Collatz Conjecture

> The Collatz Conjecture (or the "3n+1" problem) is one of the most famous unsolved mysteries in mathematics.

![# The Simplest Math Problem No One Can Solve - Collatz Conjecture](https://www.youtube.com/watch?v=094y1Z2wpJg)

It's a simple set of rules for any positive Integer:

1. Pick any number
2. If the number is odd, multiply it by 3 and add 1(3n+1).
3. If the number is even, divide it by 2.

The conjecture suggests taht no matter what number you start with, you will eventually hit a loop of 4,2,1.
While it has been tested for numbers up to roughly `\(2^{68}\)` and found be be true, nobody has been able ot prove that it works for _every_ possible number.

> "Mathematics may not yet ripe for such questions" ~ _Paul Erdős_
